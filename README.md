## 问题
### hosts
当在下载安装 `vim-plug` 时，可能会安装失败。
这可能是 `hosts` 的问题，你可以通过修改 `hosts` 文件来解决这个问题。
添加下面的代码进你的 `hosts` 文件内。
```
199.232.28.133 raw.githubusercontent.com
```
